# 🌸 DecodeLabs AI Project 2 - Iris Flower Classification using KNN

## Project Overview

This project is developed as part of the DecodeLabs Artificial Intelligence Internship (Project 2).

The objective of this project is to build a Machine Learning Classification Model using the famous Iris Dataset. The model predicts the species of an iris flower based on its sepal and petal measurements using the K-Nearest Neighbors (KNN) algorithm.

---

## Features

- Load Iris Dataset
- Explore Dataset
- Data Preparation
- Train-Test Split
- Feature Scaling using StandardScaler
- K-Nearest Neighbors (KNN) Model
- Model Training
- Prediction on Test Data
- Accuracy Evaluation
- Confusion Matrix
- Classification Report
- F1 Score
- Custom User Prediction
- Species Distribution Visualization
- Confusion Matrix Visualization

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Dataset

Dataset Used:

- Iris Dataset (Built-in Scikit-Learn Dataset)

Features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target:

- Setosa
- Versicolor
- Virginica

---

## Machine Learning Algorithm

Algorithm Used:

**K-Nearest Neighbors (KNN)**

The model classifies a flower by finding the nearest neighbors and selecting the majority class.
